# Arrow Maze 微信小游戏

一个“点击箭头折线并尝试滑出屏幕”的原创微信小游戏原型。

## 当前玩法

- 点击任意一条线，角色会沿箭头方向移动。
- 线条本身就是遮挡物，没有额外挡板。
- 如果前方没有其他线身阻挡，这条线会直接滑出屏幕并通关。
- 如果被其他线身挡住，就会失败并失去 1 条血。
- 每关有 3 条血，支持提示、音效和分享复活。

## 目录结构

- `game.js`：小游戏入口
- `js/app.js`：主循环、界面绘制和交互
- `js/config.js`：常量和视觉配置
- `js/utils.js`：几何与路径工具
- `js/sound.js`：轻量音效管理
- `js/levels.js`：手工关卡与程序关卡生成

## 运行方式

1. 打开微信开发者工具
2. 选择“小程序 / 小游戏”项目
3. 导入目录 `D:\personWork\arrow-maze`
4. AppID 可先使用测试号或 `touristappid` 本地预览
